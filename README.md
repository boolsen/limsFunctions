# limsFunctions

This repository contains the LIMS Basic function definition source text and a small static web app for searching those definitions.

## Updating the function search data

The app uses `app/functions.json` as its search database. That file is generated from `limsFunctions_text_format.txt`.

If you update `limsFunctions_text_format.txt`, regenerate the JSON with:

```bash
python3 app/generate_functions_json.py
```

This will recreate `app/functions.json` from the updated source text.

## Deployment

The `app/` folder is the Netlify deploy root. The root `netlify.toml` file sets `publish = "app"` so Netlify serves the static app from that directory.

## Notes

- `parse_lims_functions.py` contains the parser logic used by the generator.
- The generated SQLite database is no longer required for the deployed web app.

## Deployment

The app is deployed on netlify, with the name limsfunctions. After regenerating the json file as described above trigger a deploy on netlify to get updates.

The text file is generated by downloading the lims functions page in labwares manual and copying all text from FSnSH2.html
